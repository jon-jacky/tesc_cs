Concepts of Computing, Section II

Jon Jacky

Concepts of Computing is a 10 week, 8 credit introduction for novices with little computing experience and no programming experience. There is a lecture/readings/seminar component where students read five current books on technical and social aspects of computing and comment on them in short writing assignments and in discussion. There is a computing/lab component where students create a web site using HTML and write some small programs using Visual Basic.
