Operating Systems

An operating system is a collection of programs that work together to provide the services of a computer in a convenient form to users and programmers.

In Fall term, we will have become familiar with the ubiquitous Windows operating system. In Winter, we will learn and use Linux, a popular modern version of the vastly influential Unix operating system. In addition to using Windows and Linux, we will study the strategies their designers adopted for organizing large software systems.

Many computer specialists find Unix (in its various versions such as Linux etc.) to be the most productive operating system for programming and innovative computing. Many of the influential developments of the last thirty years were created or first popularized on Unix systems.

Linux is a good choice for learning about operating systems because its design is apparent and largely exposed to its users. It is unusually well described in many books and an immense store of on-line material (including all the source code).

Networked Applications

When you use a computer these days, it is likely that you are actually working with several different computers that communicate over a network. For example, when you use a web browser your computer communicates with remote server computers that send the content you see in the web pages.

We will study how networked applications are constructed and experiment with building some ourselves.

Textbooks

I haven't chosen the textbooks yet. I am considering these. You should not run out and buy these yet!

SAM'S Teach Yourself Unix in 10 Minutes, William Ray, ISBN 0-672-31523-8, 210 pages, about $13. (Useful if you are a total newbie, but actually takes more than ten minutes).

Linux in a Nutshell, Ellen Siever, ISBN 1-56592-585-8, O'Reilly, 612 pages, about $25. (Encyclopedic reference, highly recommended)

Networked Applications: A Guide to the New Computing Infrastructure, David G. Messerschmitt, 1999, Morgan Kaufmann, About $40. (Good topic coverage but perhaps not sufficiently technical)

Not Just Java, Peter Van der Linden, About $40. (Covers many of the same topics as Messerschmitt, a bit more technical, heavy on the pro-Java, anti-Microsoft advocacy)

I am still looking for a good introductory technical book on networks, with project ideas.
